# Xtension v0.4.16

- Ships the Windows bridge as a signed installer that creates the `XtensionBridge` service automatically.
- Runs the bridge silently in the background with no console window after installation.
- Cleans up the legacy startup task used by earlier local fallback installs.
- Keeps Codex/Grok/Gemini/Claude detection under the installing Windows user profile while the service runs under Windows.

- `xtension-edge-v0.4.16.zip`
- `xtension-chrome-v0.4.16.zip`
