# Xtension v0.4.9

- Adds a localized final line to generated PDFs after the source URL.
- Links the X sign-up text to `https://x.com/` and links the exported `@account` to its X profile.
- Replaces the PDF menu icon with a monochrome document icon that fits the X/Twitter menu style.
- Keeps the validated multicolor double-struck X application logo in all browser packages.
- Rebuilds Chrome, Edge, and Firefox archives from the same committed assets.
