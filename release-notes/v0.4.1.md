# Xtension v0.4.1

- Adds WebExtension localization for 21 locales: `ar`, `cs`, `da`, `de`, `en`, `es`, `fi`, `fr`, `hi`, `it`, `ja`, `ko`, `nl`, `no`, `pl`, `pt_BR`, `pt_PT`, `ro`, `sv`, `tr`, `zh_CN`.
- Localizes extension metadata, the X/Twitter menu action, export progress messages, image fallback text, video notes, and PDF source labels.
- Updates public documentation and generated promotional visuals to English.
- Keeps PDF export behavior unchanged from v0.4.0.
