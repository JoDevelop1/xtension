# Xtension v0.3.0

Première release publique utilisable en sideload en attendant la publication sur les stores.

## Téléchargements

- Edge : `xtension-edge-v0.3.0.zip`
- Chrome, Brave, Vivaldi, Opera : `xtension-chrome-v0.3.0.zip`
- Firefox : `xtension-firefox-v0.3.0.zip`
- Checksums : `SHA256SUMS.txt`

## Installation rapide

1. Télécharge le zip correspondant à ton navigateur.
2. Décompresse-le dans un dossier stable.
3. Charge le dossier décompressé depuis la page extensions du navigateur.

Instructions détaillées : voir `RELEASES.md`.

## Nouveautés

- Nom public : **Xtension**.
- Logo `×` mathématique stylisé, distinct du logo officiel X.
- Export PDF direct depuis les articles X/Twitter.
- Intégration des images média dans le PDF.
- Packages séparés Edge / Chrome-Chromium / Firefox.
