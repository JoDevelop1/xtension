# Xtension v0.3.0

First public release usable through sideloading while store publication is pending.

## Downloads

- Edge: `xtension-edge-v0.3.0.zip`
- Chrome, Brave, Vivaldi, Opera: `xtension-chrome-v0.3.0.zip`
- Firefox: `xtension-firefox-v0.3.0.zip`
- Checksums: `SHA256SUMS.txt`

## Quick Installation

1. Download the zip matching your browser.
2. Extract it into a stable folder.
3. Load the extracted folder from the browser extensions page.

Detailed instructions: see `RELEASES.md`.

## New

- Public name: **Xtension**.
- Stylized mathematical `x` logo, distinct from the official X logo.
- Direct PDF export from X/Twitter articles.
- Media images embedded in the PDF.
- Separate packages for Edge, Chrome/Chromium, and Firefox.

Xtension is an independent project and is not affiliated with X Corp. or Twitter.
