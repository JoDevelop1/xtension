# Xtension v0.4.6

- Adds hidden PDF metadata with the active Xtension version in `Creator` and `Producer`.
- Keeps the v0.4.5 Helvetica-compatible text metrics for clean PDF body justification.
- Makes stale browser content scripts easier to detect when a regenerated PDF still looks unchanged after an update.

- `xtension-edge-v0.4.6.zip`
- `xtension-chrome-v0.4.6.zip`
- `xtension-firefox-v0.4.6.zip`
