# Xtension v0.4.0

- Ajoute **Télécharger en PDF** aux menus de tweets X/Twitter.
- Exporte les threads en regroupant les messages contigus du même auteur.
- Arrête l'extraction du thread avant les réponses publiées par d'autres comptes.
- Inclut les tweets cités avec leur texte et leurs médias disponibles.
- Ajoute les aperçus vidéo dans le PDF avec une mention pour lire la vidéo depuis le tweet source.
- Conserve l'export des articles long-form avec texte structuré et images.
