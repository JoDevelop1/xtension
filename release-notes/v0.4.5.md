# Xtension v0.4.5

- Fixes PDF body text justification by measuring text with Helvetica-compatible PDF character widths.
- Keeps the last line of each paragraph naturally ragged while aligning non-final prose lines to the right margin.
- Keeps headings, media, embedded cards, and centered content in their existing layout.
