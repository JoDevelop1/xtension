# Xtension

## Short summary

Download X/Twitter articles, tweets and threads as PDF files with images.

## Description

Xtension adds a focused action to the X/Twitter `...` menu: **Download as PDF**.

When you open the `...` menu on an X article, tweet or thread, the extension detects the relevant content, extracts text, structure, quoted tweets and media images, then generates a local PDF in your browser. For threads, Xtension groups contiguous posts from the same author and stops before replies from other accounts. The browser opens a **Save as** dialog so you can choose the file name and destination folder.

Key features:

- Direct PDF export from an X/Twitter article, tweet or thread menu.
- Preserves article structure: headings, paragraphs, lists and quotes.
- Detects same-author threads without capturing every reply.
- Includes quoted tweets with their available content.
- Embeds media images in the PDF.
- Local generation without an external service.
- Runs only within the X/Twitter domain scope and associated media domains.
- Works with Edge, Chrome, Brave, Vivaldi, Opera and Firefox using the matching package.

Permissions:

- `x.com` / `twitter.com`: add the action to the X/Twitter menu.
- `pbs.twimg.com`: fetch public X/Twitter images, avatars, card images and video preview thumbnails referenced by the selected content.

The extension does not collect personal data, does not send content to a third-party server and does not modify your X/Twitter account.
